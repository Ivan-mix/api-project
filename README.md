# Recipe App

A simple recipe search app built with HTML, CSS, and JavaScript.

## Features

- Search meals by name with TheMealDB API
- View recipe cards with area and category capsules
- Open a recipe popup with ingredients and instructions
- Responsive layout for desktop and mobile

## Files

- `index.html` - page structure
- `style.css` - UI design and responsive styles
- `script.js` - recipe search and popup behavior

## Run locally

1. Open `index.html` in your browser.
2. Search for a meal name such as `chicken`, `beef`, or `pasta`.

## Notes

- Recipe data comes from [TheMealDB](https://www.themealdb.com/api.php).
- Category capsules use muted custom colors for a cleaner look.
